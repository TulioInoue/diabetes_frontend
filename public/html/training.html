Motivation
¶

The Early Stage Diabetes Risk Prediction Dataset is a critical resource for healthcare analytics, containing clinical data from patients who have recently been diagnosed with diabetes or exhibit significant symptoms. In the medical field, early detection is paramount because undiagnosed diabetes leads to severe long-term complications, such as cardiovascular disease, kidney failure, and permanent nerve damage. By identifying high-risk individuals through symptoms like polyuria and polydipsia, healthcare providers can intervene early, potentially reversing or managing the condition before it escalates.

The goal of this project is to develop a robust classification model capable of predicting the presence of diabetes with high precision. To achieve this, a rigorous data cleaning and preprocessing pipeline is essential. Refining the raw dataset — handling categorical variables, ensuring consistency, and scaling features — is vital to eliminate bias and noise. Ultimately, this classification project aims to provide a reliable diagnostic support tool, leveraging machine learning to enhance clinical decision-making and improve patient outcomes through timely and accurate risk assessment.

4.2) GridSearch¶

In [13]:
for model in models:
    
    pipeline = Pipeline(
        steps = [
            ("pre_processing", pre_processing),
            ("model", model["model"])
        ]
    )

    estimator = GridSearchCV(
        estimator = pipeline,
        param_grid = model["paramns"],
        n_jobs = -1,
        cv = 5,
        error_score = "raise",
        scoring = "f1",
        refit = True
    )

    estimator.fit(
        X = model['x_train'],
        y = model['y_train']
    )

    joblib.dump(
        filename = f"./models/tests/{model['name']}.joblib",
        value = estimator.best_estimator_
    )

trained_models = []

for archive in Path("./models/tests/").iterdir():

    trained_models.append(
        {
            "name": archive.name.replace(".joblib", ""),
            "model": joblib.load(
                filename = archive
            )
        }
    )

fig, axs = plt.subplots(
    ncols = 4,
    nrows = 2,
    figsize = (18, 7),
    gridspec_kw = {
        "hspace": 0.4
    }
)

axs = axs.flatten()

metrics = {
    "model": [],
    "f1": [],
    "accuracy": [],
    "recall": [],
    "time": []
}

for index, model in enumerate(trained_models):

    start_time = time()
    prediction = model["model"].predict(
        X = X_test
    )
    end_time = time()

    metrics["model"].append(model["name"])
    metrics["time"].append(end_time - start_time)
    metrics["f1"].append(
        f1_score(
            y_true = y_test,
            y_pred = prediction
        )
    )
    metrics["accuracy"].append(
        accuracy_score(
            y_true = y_test,
            y_pred = prediction        
        )
    )
    metrics["recall"].append(
        recall_score(
            y_true = y_test,
            y_pred = prediction
        )
    )

    plots.plot_confusion_matrix(
        ax = axs[index],
        y_true = y_test,
        y_pred = prediction,
        title = {
            "text": model["name"]
        }
    )

plt.suptitle(
    t = "MODELS PERFORMANCE",
    fontsize = 24,
    fontweight = "bold",
    fontfamily = "arial"
)
plt.show()
